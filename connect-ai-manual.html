B

Jay Jung의 핵심 철학

1

AI 소비자 → AI 생산자

ChatGPT에 돈 내고 질문만 하는 게 아니라, 내 로컬에서 AI를 직접 운영해 비즈니스에 쓰는 것. 비용 0원, 데이터 외부 유출 없음.

2

1인 기업 + AI = 유니콘 가능성

2026년 글로벌 트렌드: AI를 팀원으로 쓰는 솔로프리너(1인 창업자)가 기존 10명 팀 이상의 생산성을 낼 수 있는 시대. Jay Jung의 solopreneur-ai-agents 레포가 이 철학의 실체.

3

Agent University (A.U) 시스템

단순 채팅 AI가 아니라, 자율적으로 파일 생성·편집·삭제·명령 실행·Git 자동 동기화까지 하는 에이전트 시스템. "시키면 알아서 한다"가 핵심.

4

P-Reinforce 아키텍처

단순 답변 → 자율 행동(Autonomous Gardener). AI가 스스로 지식을 정리하고, 코드를 쓰고, Git에 올리는 루프를 만드는 구조.

E

초기 세팅 & 설정

1

Agent University (A.U) 브레인 팩 설치

VS Code에서 포트 4825로 브레인 팩 접근 가능. ~/.connect-ai-brain 폴더에 지식 베이스 저장됨.

2

작업 폴더 열기

VS Code → File → Open Folder → 작업할 폴더 선택. Connect AI는 이 폴더 내 파일을 자동으로 읽고 편집함.

3

시스템 프롬프트 설정

Connect AI 채팅창 상단 설정에서 System Prompt 입력. 회사 맥락, 역할, 규칙을 여기에 넣으면 매번 자동 적용됨.

4

Auto-Git Sync 활성화 (선택)

Git 연동 시 파일 변경사항을 자동 commit/push. GitHub 토큰 필요. 협업·버전관리에 유용.

1

AI 팀원 역할 분담 설계

사람 직원 대신 AI 에이전트에게 역할 부여. 마케터 AI, 개발자 AI, 분석가 AI, 영업 AI를 각각 Connect AI 프롬프트로 설계.

실전 팁 & 노하우

💡 팁 1: 파일 드래그로 컨텍스트 주입

엑셀, 텍스트, JSON 파일을 Connect AI 입력창에 드래그하면 AI가 해당 파일 내용을 읽고 답변. 인플루언서 DB 파일 넣고 "A등급 추려줘" 가능.

💡 팁 2: 역할 페르소나 고정

시스템 프롬프트에 "당신은 면세점 MD 8년 경력 박상권의 AI 직원입니다"처럼 구체적 역할 지정. 답변 품질 크게 향상.

💡 팁 3: 결과물 반복 개선

"더 짧게", "더 친근하게", "2번째 문단 다시 써줘"처럼 이어서 수정 지시. 처음 결과물보다 3~4번 수정한 게 훨씬 좋음.

💡 팁 4: 배치 작업

"인플루언서 10명에게 보낼 DM 10개 한번에 써줘 - 계정명 목록: [목록]" → 한 번에 대량 생성 가능.

💡 팁 5: .cursorrules 활용

프로젝트 폴더에 .cursorrules 파일 생성 → 해당 폴더 열 때마다 자동으로 규칙 적용. 브랜드앤유 폴더에 규칙 고정 가능.

💡 팁 6: 더 강력한 모델로 업그레이드

Qwen2.5-1.5B → Qwen2.5-7B로 바꾸면 답변 품질 크게 향상. LM Studio에서 모델 검색 후 다운로드 (4.5GB). RAM 8GB+ 권장.
